Detailed Description

Gazebo camera subscriber with real-time station outline and opening detection.

Subscribes to Gazebo image topics via gz.transport, runs an OpenCV-based detection pipeline on every frame, and displays the annotated results in OpenCV windows.

Detected objects:

  • Station: large rectangular contours (station outlines), drawn in cyan-yellow.
  • Station opening: highest-density cluster of hex-angle Hough lines, drawn in orange.
