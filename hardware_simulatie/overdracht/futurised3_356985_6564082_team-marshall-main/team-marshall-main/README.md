# Brandweerrobot Flip

Welkom bij de repository van Team Marshall voor Brandweerrobot Flip. Dit project bevat de code, simulatiebestanden en documentatie voor de autonome brandweerrobot in de Gazebo-simulatieomgeving.

## Documentatie

Alle documentatie en de bijbehorende website zijn lokaal in deze repository beschikbaar. U kunt de documentatie op twee manieren bekijken:

### Via de interactieve HTML-website (Aanbevolen)
Binnen deze repository is een grafische website gebouwd waarin alle documentatie overzichtelijk is samengebracht, inclusief afbeeldingen en media. 

Om deze website te openen:
1. Navigeer naar de hoofdmap van dit project.
2. Open het bestand **`main.html`** in een webbrowser naar keuze.

* [Documentatie](Documentatie/website/HTML) – Website

> ### Belangrijke opmerking bij gebruik
> Mocht de HTML-pagina onverhoopt vastlopen of crashen tijdens het navigeren, keer dan simpelweg terug naar de hoofdmap (main) van de repository en open het bestand `main.html` opnieuw om de sessie te herstarten.

### Direct via de losse Markdown-bestanden (meer technisch uitgelegd)
Voor directe inzage binnen de mappenstructuur is de volledige documentatie beschikbaar in de map `documentatie/`. 

Gebruik de onderstaande link om direct naar het gewenste onderdeel te navigeren:

* [Documentatie](Documentatie) – Alle documentatie van Team Marshall
