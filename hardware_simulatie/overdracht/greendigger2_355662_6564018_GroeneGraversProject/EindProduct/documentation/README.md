# README Documentation
This folder contains research of the project and the agreements we made with the GreenDigger about functionalities. This section also includes test reports for testing the functionality of various aspects. We have created one for Vision and Gazebo.

**Vision**
This section discusses whether the camera is functioning properly, whether the computer vision system is working as intended, and whether users are satisfied. These various issues span a broad spectrum, allowing us to cover all aspects within a given topic. Since we have chosen to consider computer vision as a whole, we are referring to both the camera that takes photos and OpenCV, which is designed to recognize “trees.” 

**Gazebo**
It is also important that the gazebo undergoes testing. These tests primarily focus on the PID and the arm, specifically regarding accuracy and range. We don’t want to be faced with surprises from a product that turns out not to meet the requirements. For this reason, we have decided to test in advance to ensure that all requirements are met.
