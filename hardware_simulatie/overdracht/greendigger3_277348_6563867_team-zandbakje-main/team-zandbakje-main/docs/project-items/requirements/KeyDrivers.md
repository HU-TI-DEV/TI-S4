### Key Driver Analysis

`Authors: Ocarian, Replitard, GrijzePanda, ShelleKjell` 

| User/Stakeholder              | Functional objective                                                                     | Key Design Driver | Rationale/Notes                                                                                                                                                       |
| ----------------------------- | ---------------------------------------------------------------------------------------- | ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Students                      | Understand how to use simulation software.                                               | Educational       | To learn how to effectively work in a team, work in a simulated environment.                                                                                          |
| GreenDigger                   | Test ideas without material costs.                                                       | Monetary          | Simulating costs less then building, deploying and running a whole project. There is also the benefit of simulations being able to run in tandem for time efficiency. |
| Teachers                      | Each student has to build something 'intelligent', which the teacher has to sign off on. | Burden of proof   | Each student has to show their capability in said solution.                                                                                                           |
| GreenDigger budget department | Students with their lack of experience offer a unique industry perspective.              | Innovative        | Comes with unique solutions to given problems.                                                                                                                        |
| Property developers           | Make the soil valuable again so it can be used.                                          | Monetary          | By making vegetation in the desert, the life of locals and businesses will improve.                                                                                   |
| Future students               | Setup a project ready for future development.                                            | Sustaintable      | The project has to be documented and carried over for future use.                                                                                                     |
