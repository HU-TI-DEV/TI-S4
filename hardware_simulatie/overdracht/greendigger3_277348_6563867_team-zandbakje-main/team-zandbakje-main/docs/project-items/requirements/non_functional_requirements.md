## Non-Functional Requirements

`Authors: Ocarian, Replitard, GrijzePanda, ShelleKjell` 

| Name Requirements | Feature                                        | Rationale                                                                                                                                                  | Priority    | Functional<br>Requirement |
| ----------------- | ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- | ------------------------- |
| NF-001            | status reporting using Gazebo Topics           | To measure what the digger is seeing in the simulation. Such as its location relative to the 0,0,0 point, status of the sensors, output of the sensors.    | Must Have   | F-001                     |
| NF-002            | Local Obstacle Detection using LIDAR           | To detect obstacles that are greater then X % of the green digger's wheel basis in height within a X meter radius.                                         | Must Have   | F-002                     |
| NF-003            | Localization compared to the world origin.     | To know the position of the green digger in the simulation.                                                                                                | Should Have | F-003                     |
| NF-004            | Obstacle Avoidance using locomotion + LIDAR    | When an obstacle is detected, the digger will drive itself around/away from it, keeping a distance of at minimum X meters.                                 | Should Have | F-004                     |
| NF-005            | Local Planning/Control without autonomy        | The digger follows a path of key points of interest, and follows a standard digging pattern.                                                               | Should Have | F-005                     |
| NF-006            | With a simple rig, simulate digging patterns.  | To test the viability of NF-005.                                                                                                                           | Must Have   | F-006                     |
| NF-007            | stance compenstation using sensors             | On uneven terrain, the digger should still be able to dig a half circle in the required depth.                                                             | Should Have | F-007                     |
| NF-008            | Excavation Fallback Behavior - Skip Excavation | The digger skips a section of land it has to plot if an obstacle is detected using LIDAR.                                                                  | Should Have | F-008                     |
| NF-009            | Actuator control using keyboard.               | In gazebo, the digger must be able to be controlled manually using key-presses from a keyboard. Letters ranging from qwertyuiopasdfghjklzxcvbnm1234567890. | Should Have | F-009                     |
| NF-010            | Digger Path Planning                           | The digger is able to make a path by itself without points of interest, and follow this path.                                                              | Could Have  | F-010                     |
| NF-011            | Locomotion with wheels.                        | The digger has wheels so it can drive around.                                                                                                              | Must Have   | F-011                     |
| NF-012            | Object Recgonition using LIDAR                 | The digger can see the difference between a tree, hole, rock and pebble.                                                                                   | Could Have  | F-012                     |
