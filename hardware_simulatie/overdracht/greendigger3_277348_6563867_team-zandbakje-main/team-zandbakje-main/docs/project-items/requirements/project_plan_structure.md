## 2. Project Objective
For this project we intend to make a simulation for the GreenDigger using Gazebo and the documents provided by GreenDigger. This simulation has a moving chassis, a digging arm, sensors to detect the environment.

---
## 3. Scope
Within the scope are the following deliverables :
- Object Detection (LIDAR)
- Digging Patterns (based on sensor data)
- Locomotion (driving with wheels)
- *Optional* Pathfinding (smart systems)
- *Optional* Object Recognition (camera)

---
## 4. Deliverables
- Documentation for future development
- Various .SDF files that work with Gazebo 10.0.0
- Demo Presentation

---
## 5. Meetings & Updates
Meetings will be held once a week in various forms in the initial stages of the project. After this, the meetings will be held less frequent and will be for progress updates and *light* steering of the project (once per sprint).

---
## 6. Risks
- Unclear requirements from GreenDigger.
- Unclear communication between GreenDigger and the Team.
- Gazebo limitations.
- Unforeseen circumstances (illness, unavailability of team, teachers, etc).
